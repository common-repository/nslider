== Installation ==
This section describes how to install the plugin and get it working.

e.g.

1. Upload `NSlider plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Place [nslider-shortcode] sortcode in your any page or post.
